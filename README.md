# 🛠️ Engineer Control Panel
**Developed by: vRMZ1**

بوت احترافي للتحكم في الكمبيوتر عن بُعد عبر تليجرام، مصمم لمراقبة أداء النظام وتنفيذ المهام الأساسية بسهولة وأمان.

## 🚀 المميزات:
* 📊 **Status:** مراقبة استهلاك الـ CPU و RAM في الوقت الفعلي.
* 📸 **Screen:** التقاط صورة حية للشاشة وإرسالها فوراً.
* 📂 **Files:** جلب أي ملف من الجهاز عبر المسار.
* 🔒 **Security:** حماية كاملة؛ البوت لا يستجيب إلا لمعرف (Admin ID) محدد.

## 🛠️ المتطلبات:
* Python 3.x
* المكتبات: `python-telegram-bot`, `psutil`, `pyautogui`

## ⚙️ التشغيل:
1. ضع التوكن الخاص بك في ملف `bot1.py`.
2. قم بتشغيل الملف باستخدام الأمر: `python bot1.py`.
